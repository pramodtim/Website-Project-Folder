Portfolio Website - Pramod Timalsina

How to use:
1. Download and unzip the folder.
2. Open the folder in VS Code (File -> Open Folder).
3. Open index.html in the browser (right-click -> Open with Live Server or open directly).
4. To enable contact form, sign up at https://formspree.io, create a form and replace the action URL in contact.html with your Formspree endpoint (https://formspree.io/f/xxxxx).
5. Replace assets/images/profile.jpg with your real profile photo (keep dimensions square, recommended 400x400).
6. Replace resume/Pramod_Timalsina_CV.pdf with your real CV if desired.
7. To publish: push to GitHub and enable GitHub Pages or deploy to Netlify (drag-and-drop).

Notes:
- Fonts are loaded from Google Fonts in the HTML (Poppins + Roboto).
- Theme toggle stores preference in localStorage.
